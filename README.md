# Hello World

## C
- compile
```
gcc HelloWorld.c -o HelloWorld
```

- run
```
./HelloWorld
```

## C++
- compile
```
gcc Helloworld.cpp -o HelloWorld -lstdc++ 
```

- run
```
./HelloWorld
```

## Java
- compile
```
javac HelloWorld.java
```

- run
```
java HelloWorld
```

## Python
- run
```
python HelloWorld.py
```

## PHP
- run
```
php HelloWorld.php
```

## NodeJS
- run
```
node HelloWorld.js
```
